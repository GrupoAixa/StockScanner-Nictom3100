CÓMO USAR
─────────────────────────────────────────────
1. Doble clic en "INICIAR_StockScanner.bat"
2. ¡Listo! No necesitás instalar nada más.

REQUISITO ÚNICO
─────────────────────────────────────────────
Python 3.8 o superior (gratis, viene en Windows 11)
→ Verificar: abrí CMD y escribí: python --version
→ Si no está: https://www.python.org/downloads/
  (marcá "Add Python to PATH" al instalar)

LECTOR NICTOM 3100
─────────────────────────────────────────────
Conectá el lector por USB antes de abrir el programa.
Funciona en modo HID (teclado) — sin drivers especiales.
En la pantalla "Escanear", simplemente apuntá y disparé.

ARCHIVOS GENERADOS
─────────────────────────────────────────────
stockscanner.db → base de datos local (no borrar)

FUNCIONES
─────────────────────────────────────────────
◉ Escanear   → Registrar entradas/salidas/ajustes
☰ Inventario → Ver stock completo, buscar, exportar CSV
⏱ Historial  → Todos los movimientos registrados
＋ Productos  → Agregar/editar/eliminar productos
⚠ Alertas    → Productos por debajo del stock mínimo
